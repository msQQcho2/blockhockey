## 使用フレームワーク/ビルド方法

[Siv3D](https://siv3d.github.io/ja-jp/)により作成

blockhockey.slnをVisual Stdio Code上でプロジェクトとして開いてデバックを開始すればexeファイルが作成される。



## ディレクトリ構成

~~~
blockhokey
├─blockhockey.sln (プロジェクトファイル)
├─README.md (ゲーム説明)
└─blockhockey (プロジェクトフォルダ)
   ├─Main.cpp (プログラムファイル)
   └─App (アプリケーションフォルダ)
     ├─config.csv (設定ファイル)
     └─images (画像フォルダ)
~~~

その他、Siv3Dが動作するために必要なファイル類が存在している。
